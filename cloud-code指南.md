## 一、Claude Code相关说明

### 1. 适用场景

Claude Code价格偏高，但在特定场景下使用更划算，简单任务用它则浪费成本。

- Thank you.适合用的情况
  - 一次性处理几百到几千行代码，做架构梳理、找Bug、模块重构、跨语言迁移
  - 需求文档与代码一并处理，按需求改代码、补全逻辑、生成注释文档
  - 批量生成完整API、页面、工具类等完整功能模块
  - 代码审查、安全审计，排查复杂逻辑漏洞与性能隐患
  - 旧项目升级、技术债清理，一键生成新版代码
  - 编写支付、订单、权限等高可靠性生产代码
- 不适合用的情况
  编写简单函数、小工具、基础页面、简单调试等，用DeepSeek‑Coder、Llama 3.3 70B等低价/免费模型即可，成本仅为Claude Code的1/10至1/50。

### 2. 200k上下文是什么意思

200k上下文指AI可一次性处理的文字长度，约等于12万字中文，直观场景如下：

- 一次性放入整套React组件库（约5000-8000行代码），完整理解项目结构
- 同时上传3000行代码+1万字需求文档，不丢失上下文直接给出重构方案
- 完整处理2000行Vue2页面并一键转为Vue3，无逻辑断裂
  普通小模型需分段处理，易出错且效率低。

### 3. 便宜购买Claude Code Token的正规渠道

- 官方渠道
  订阅Pro（20美元/月）、Max（100美元/月）或直连API，稳定但价格高；可使用提示缓存减少重复计费，非高峰时段可能有额度优惠。
- OpenRouter（推荐）
  价格比官方低10%-20%，新用户有免费额度，充值满额可享额外赠送，国内使用稳定。
- 国内正规中转API
  0011AI、API易等，价格为官方的40%-70%，支持支付宝/微信，国内直连稳定；阿里云百炼、腾讯云MaaS适合企业使用，合规可开票。
- 严禁使用渠道
  淘宝/闲鱼共享号、逆向API等超低价渠道，极易封号、泄露代码，服务商跑路风险高。

### 4. 接入Trae AI编辑器的方法

Trae基于VS Code构建，完全兼容Claude Code，有两种接入方式：

- 安装官方插件
  打开Trae扩展面板，搜索Claude Code官方插件安装，登录并绑定API Key即可使用完整功能，适合重度开发。
- 原生添加模型
  进入Trae设置-模型，选择Anthropic服务商，填入API Key，选用Sonnet或Opus模型，保存后一键切换使用，适合日常快速开发。
- 省钱建议
  日常轻量开发用Trae内置免费模型，复杂任务再切换Claude Code，降低Token消耗。

## 二、OpenRouter.ai使用说明

### 1. 作用

它是AI大模型统一调用入口，一个密钥即可访问全球数百个主流模型，兼容OpenAI代码，自动实现负载均衡与成本控制。

### 2. 快速使用步骤

1. 邮箱/GitHub注册账号
2. 充值额度或使用标free的免费模型
3. 在Keys页面生成API Key并妥善保存
4. 替换代码中的接口地址与密钥，修改模型名称即可调用

### 3. 免费与付费模型

- 免费模型
  无需绑卡，注册即用，有每日请求限制；热门包括阶跃星辰、Llama 3.3 70B、千问3.6 Plus预览版等。
- 付费模型
  按每百万Token计费，顶级模型如GPT‑4o、Claude 3 Opus，高性价比模型如DeepSeek Coder、Llama 3.1 70B，轻量模型价格极低，适合批量任务。
