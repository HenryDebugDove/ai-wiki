# 代码审查大师

## 技能概述

代码审查大师是一个专业的 JavaScript/TypeScript 代码审查工具，能够全面分析代码质量，发现潜在问题，并提供详细的修复建议。

## 审查维度

### 1. ESLint 规范检查
- 检查代码是否符合 ESLint 最佳实践
- 识别常见的代码风格问题
- 检查未使用的变量和导入
- 验证代码格式的一致性

### 2. 安全漏洞检测
- SQL 注入风险
- XSS 攻击风险
- 敏感信息泄露
- 不安全的 API 调用
- 依赖包安全漏洞

### 3. 性能瓶颈分析
- 循环性能问题
- 内存泄漏风险
- 不必要的重复计算
- 异步操作优化
- DOM 操作优化

### 4. 注释覆盖率
- 函数注释完整性
- 复杂逻辑说明
- 参数和返回值文档
- 代码可读性评估

## 输出格式

### 问题清单
为每个发现的问题提供：
- **问题类型**：ESLint/安全/性能/注释
- **严重程度**：高/中/低
- **问题描述**：详细说明问题所在
- **代码位置**：文件名和行号
- **影响范围**：可能造成的后果

### 修复代码
- 提供修复后的完整代码
- 标注修改的部分
- 说明修改原因

### 综合评分
- **ESLint 规范**：0-100 分
- **安全性**：0-100 分
- **性能**：0-100 分
- **注释质量**：0-100 分
- **综合评分**：0-100 分

## 审查流程

1. **代码分析**：解析代码结构，识别语法和语义
2. **规则检查**：应用 ESLint 规则和安全检查
3. **性能分析**：识别性能瓶颈和优化机会
4. **注释评估**：检查注释覆盖率和质量
5. **问题汇总**：生成详细的问题清单
6. **修复建议**：提供具体的代码修复方案
7. **综合评分**：给出整体代码质量评分

## 评分标准

### 优秀 (90-100分)
- 完全符合 ESLint 规范
- 无安全漏洞
- 性能优化良好
- 注释完整清晰

### 良好 (80-89分)
- 基本符合规范，少量小问题
- 无严重安全漏洞
- 性能表现良好
- 注释基本完整

### 中等 (70-79分)
- 存在一些规范问题
- 有少量安全风险
- 存在性能优化空间
- 注释不够完整

### 较差 (60-69分)
- 规范问题较多
- 存在明显安全漏洞
- 性能问题明显
- 注释严重不足

### 差 (0-59分)
- 严重违反规范
- 存在严重安全漏洞
- 性能问题严重
- 几乎没有注释

## 最佳实践建议

1. **定期审查**：建议在代码提交前进行审查
2. **持续改进**：根据审查结果持续优化代码
3. **团队协作**：建立团队代码规范，统一审查标准
4. **自动化**：将审查集成到 CI/CD 流程中

## 注意事项

- 本工具提供的是静态代码分析，不能替代人工审查
- 某些性能问题可能需要运行时分析才能发现
- 安全审查应结合实际业务场景进行评估
- 注释质量需要结合代码复杂度综合判断

## 示例

### 输入代码
```javascript
function getUserData(userId) {
  const data = db.query('SELECT * FROM users WHERE id = ' + userId);
  return data;
}
```

### 审查结果
**问题清单**：
1. [安全-高] SQL 注入风险：使用字符串拼接构建 SQL 查询
2. [注释-中] 函数缺少注释说明
3. [性能-低] 建议使用参数化查询

**修复代码**：
```javascript
/**
 * 根据用户ID获取用户数据
 * @param {number} userId - 用户ID
 * @returns {Object} 用户数据
 */
function getUserData(userId) {
  const data = db.query('SELECT * FROM users WHERE id = ?', [userId]);
  return data;
}
```

**评分**：
- ESLint 规范：85分
- 安全性：60分
- 性能：90分
- 注释质量：70分
- **综合评分：76分**
